# XERO TODO App

A todo app built with React, Vite, Tailwind CSS, and Material UI dialogs.

## Features

- Create, complete, edit, and delete tasks
- Persist tasks in localStorage
- Responsive dark UI

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Material UI

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```
